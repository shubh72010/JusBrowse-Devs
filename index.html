Research Notes

Developer notes, experiments, and observations from JusBrowse development.

WebView Performance vs Custom Renderer

Comparing Android WebView rendering speed to Chromium-based custom rendering. JusBrowse uses WebView for simplicity and APK size, with targeted optimizations via WebSettings flags. Tests show ~15% faster TTI than stock browser on low-end devices with aggressive cache settings.

Performance

Advanced Fake Mode ‚Äî Digital Persona Spoofing

Deep hardware camouflage including User-Agent spoofing, screen resolution matching, hardware concurrency, WebGL GPU DNA (Mali vs Adreno), canvas fingerprint noise, WebRTC shield for IP leak prevention, and clock skew timing attack protection.

Privacy

Tracker Blocking via Hosts List Injection

Research into blocking trackers at the WebView request level using shouldInterceptRequest(). Testing custom blocklist formats (uBlock origin-style) vs simple hosts file. Results indicate ~70% tracker reduction with minimal performance impact on mid-range Android devices.

Privacy

Tab Memory Management

Investigating WebView memory leaks when destroying and recreating tabs. Retaining WebView references after navigating away causes significant heap growth. Solution: custom tab pool with bounded size and LRU eviction of background tabs.
